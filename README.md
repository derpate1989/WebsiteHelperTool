# WebsiteHelperTool

Kleines Windows-Hilfstool für Website-Arbeiten.

## Funktionen

- Bilder per Drag and Drop komprimieren
- Qualität auf 100%, 75%, 50% oder 25% setzen
- Bilder als JPG speichern
- Codebloecke in einen fertigen HTML-Codeblock für die Website umwandeln (bespielsweise Wordpress)

## Download / EXE

Im Repository liegt das FIle:

```text
WebsiteHelperTool.exe
```

## Hinweis
WebsiteHelperTool ist ein kleines privates Hilfsprojekt und kein professionelles Bildbearbeitungsprogramm. Es ist bewusst einfach gehalten und auf praktische Alltagsaufgaben ausgelegt.
Das Tool wurde mit Hilfe von AI entwickelt, um wiederkehrende Website-Aufgaben einfacher und schneller zu machen.
